# claunch — ink prototype

TUI preset launcher built with [Ink](https://github.com/vadimdemedes/ink) (React for terminals).
One of three competing UI styles in the claunch bake-off.

## Run

```
npm install && npm start
```

## Non-interactive (print default command and exit)

```
npm start -- --non-interactive
```

## Keys

`↑↓` select preset · `enter` launch · `q` quit
